◆ blnDebugMode

const bool global::blnDebugMode = false

This will hold if this build is meant for debugging. While in debugging mode the program will print many more messages to the console. It is set to true if DEFINED_BUILD_MODE_PRIVATE is defined in version.h or during compiling.
